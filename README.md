## 👨🏻‍💻 Overview  
과거부터 진행 되어 온 'TOC 튀는 멘토링'은 학습법,  학업 능력 향상을 목표로 한 교내 프로그램이다.  
이 프로그램은 학기 말에 심사를 통해 우수 팀을 선정하는데, 심사 기준에 멘티들의 3-in 이수 현황도 포함된다.  
따라서 멘티들이 웹 페이지를 통해 본인들의 이수 현황을 기록하고 확인 할 수 있게 하고자 본 프로젝트를 기획하였다.  

<br/>
